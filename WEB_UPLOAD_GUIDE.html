🐱 雅思猫 - GitHub上传指南

一步一步教你将雅思猫部署到GitHub Pages

1

创建GitHub仓库

首先，创建一个新的GitHub仓库：

https://github.com/new

填写信息：

  • Repository name: ielts-cat-app
  • Description: 雅思发音学习工具
  • Public: ✅ 勾选（公开）
  • 其他选项: ❌ 全部不勾选
2

上传文件

创建成功后，直接上传文件：

https://github.com/irvingmomo/ielts-cat-app/upload

上传步骤：

  1. 打开文件夹：C:\Users\11\ielts-cat-simple\deploy\
  2. 全选所有文件（Ctrl+A）
  3. 拖拽到GitHub上传区域
  4. 填写Commit message: Initial commit: IELTS Cat web app
  5. 点击 Commit changes
3

启用GitHub Pages

启用GitHub Pages功能：

https://github.com/irvingmomo/ielts-cat-app/settings/pages

配置设置：

  • Source: Deploy from a branch
  • Branch: main
  • Folder: /(root)

点击 Save 保存设置

4

访问网站

等待1-2分钟，然后访问你的网站：

https://irvingmomo.github.io/ielts-cat-app/

💡 提示： 如果显示404错误，请等待几分钟再刷新。GitHub Pages需要时间部署。

5

备用方案

如果GitHub有问题，可以使用：

Vercel（最简单）

拖拽上传，立即部署：

本地使用

即使不部署，也可以本地使用：

# 启动本地服务器
cd C:\Users\11\ielts-cat-simple\deploy
python -m http.server 8080

# 访问 http://localhost:8080
