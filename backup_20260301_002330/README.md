# 🐱 雅思猫 (IELTS Cat)

一个免费的雅思发音学习工具，帮助您从IELTS 6.0提升到7.5分。

## 🌐 在线访问

**GitHub Pages**: https://irvingmomo.github.io/ielts-cat-app/

## ✨ 功能特点

### 🎯 核心学习功能
- **发音练习**: 使用Web Speech API进行发音学习
- **词汇库**: 200+雅思核心词汇
- **句子跟读**: 21个常用雅思句子
- **音标学习**: 43个国际音标
- **学习进度**: 自动保存学习进度

### 📱 平台支持
- 手机、平板、电脑全平台支持
- 响应式设计，适配所有屏幕
- PWA支持，可安装为桌面应用

### 🎨 用户体验
- 简洁美观的界面设计
- 实时发音反馈
- 学习统计和进度追踪
- 离线学习支持

## 🚀 快速开始

### 在线使用
1. 访问: https://irvingmomo.github.io/ielts-cat-app/
2. 点击"进入雅思猫学习应用"
3. 开始学习

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/irvingmomo/ielts-cat-app.git

# 进入目录
cd ielts-cat-app

# 使用Python启动本地服务器
python -m http.server 8080

# 访问 http://localhost:8080
```

## 📖 学习计划

### 每日学习建议
- **工作日**: 30分钟/天
- **周末**: 3小时/天
- **目标**: 3个月内从6.0提升到7.5

### 学习模块
1. **词汇发音** - 每天学习10个新单词
2. **句子跟读** - 每天练习3个句子
3. **音标学习** - 每周掌握5个音标
4. **模拟测试** - 每周一次完整测试

## 🔧 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **发音API**: Web Speech API
- **存储**: localStorage
- **部署**: GitHub Pages
- **PWA**: Service Worker, Manifest

## 📱 PWA安装

### 桌面安装
1. 在Chrome中访问应用
2. 点击地址栏右侧的"安装"图标
3. 添加到桌面

### 手机安装
1. 在Safari/Chrome中访问应用
2. 点击分享按钮
3. 选择"添加到主屏幕"
